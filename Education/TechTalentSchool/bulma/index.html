What is Bulma?

According to the Official site:

Bulma is an open source CSS framework based on Flexbox and used by more than 100,000 developers.

Bulma is:

  • Easy to learn
  • Quick to start with (only 1 file needed)
  • Can be customized (it is based on SASS)
  • There is no JavaScript required.

Bulma is simply a collection of CSS classes to help you write the HTML you want, without ever leaving the markup file.

By using descriptive and easy to memorize CSS classes, Bulma is equally easy to write and read. One can simply go through the HTML file and easily visualize the page styling by reading the CSS classes applied to the page elements and sections.

It is extremely efficient in prototyping and quickly building page layouts and web interfaces. It is also, a lightweight CSS library and the size can be trimmed down using modularization.
